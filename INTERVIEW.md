# Paul 인터뷰 노트 — 이동 중에 답하기

이동 중에 생각을 모으는 곳입니다. 실시간 반영 아님 — 모아뒀다가 한 번에 사이트에 들어갑니다.

## 사용법
1. 시간 날 때 아무 질문에나, 아래 `> ` 뒤에 편하게 적으세요. **음성입력 OK, 두서없어도 됩니다.** 몇 개든 상관없어요.
2. 답 안 한 질문은 그냥 두면 됩니다.
3. 다음에 Claude Code를 열고 **"인터뷰 업데이트"** 라고 말하면 — 답한 것만 골라 사이트(data.js)에 이야기로 엮고, 답한 질문은 이 목록에서 지우고, 새 질문을 채워 넣습니다.
4. 편집 방법 (아무거나):
   - 📱 폰: GitHub 앱 또는 브라우저에서 이 파일 열고 ✏️ → 답 쓰고 → Commit. 바로가기: `github.com/wensonjabi-p/paul-intro/edit/main/INTERVIEW.md`
   - 💬 또는 그냥 우리 대화에 답을 붙여넣어도 됩니다 (그럼 이 파일 안 거쳐도 됨).

> 팁: 질문 옆 `[...]`는 제가 어디에 연결할지 표시하는 태그예요. 신경 안 쓰셔도 됩니다.
